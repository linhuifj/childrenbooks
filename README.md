# childrenbooks

将经典历史著作改编为适合儿童阅读的绘本版本。

## 制作方式

- **文字改编**: 使用 LLM (Claude) 将原著改编为适合 8 岁儿童阅读的版本
- **插图生成**: 使用 Flux.1 schnell 模型生成水彩风格插图
- **图文对齐**: 使用 LLM Vision 自动将插图放置在最匹配的文字段落后面
- **质量控制**: 使用 LLM Vision 逐张评估插图质量，不合格的调整 prompt 后重新生成

AI 生成的插图经过多轮自动化质量检验和重新生成，但由于 AI 图像生成技术的固有局限性，部分插图仍可能存在细微瑕疵（如模糊文字、人物细节不自然等），不影响整体阅读体验。
