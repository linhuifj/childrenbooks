# childrenbooks

将经典历史著作改编为适合儿童阅读的绘本版本。原著内容深刻但对孩子来说太难懂，于是我将它们改编成少年版，配上插图，做给我儿子看。他读了之后觉得很好看，看得停不下来，这让我觉得这件事值得分享出来。

## 制作方式

1. **OCR 识别**: 下载原著 PDF，使用 OCR 技术将原文提取为文本
2. **文字改编**: 使用 LLM 将原著改编为适合 8 岁儿童阅读的版本
3. **插图生成**: 使用 Flux.1 schnell 模型生成水彩风格插图
4. **图文对齐**: 使用 LLM Vision 自动将插图放置在最匹配的文字段落后面
5. **质量控制**: 使用 LLM Vision 逐张评估插图质量，不合格的调整 prompt 后重新生成
